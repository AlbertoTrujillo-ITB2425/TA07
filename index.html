Consells per reduir el consum ♻️:

Per reduir el consum elèctric:

  • Apaga els llums quan no els necessitis.
  • Utilitza bombetes d'alta eficiència energètica (LED).
  • Desendolla els aparells electrònics quan no els facis servir.
  • Opta per electrodomèstics amb una bona etiqueta energètica.

Per reduir el consum d'aigua:

  • Redueix el temps de dutxa per estalviar aigua.
  • Repareu les fuites d'aigua per evitar malbaratar-la.
  • Utilitza aixetes amb dispositius de control de flux.
  • Recull aigua de la pluja per a ús no potable.

Per reduir el consum de materials:

  • Opta per materials reciclats sempre que sigui possible.
  • Redueix la quantitat de paper que utilitzes i opta per versions digitals.
  • Reutilitza els materials d'escola sempre que puguis.
  • Compra materials que siguin més duradors i sostenibles.

Per reduir el consum de productes de neteja:

  • Opta per productes de neteja ecològics i no tòxics.
  • Fes servir envasos reutilitzables per a productes de neteja.
  • Evita els productes amb molta quantitat de plàstic.
  • Utilitza materials naturals com el vinagre per netejar quan sigui possible.
